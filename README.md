# How to Run the Projects

## Step 1

Install Python on your system.

Download from:

https://www.python.org/

---

## Step 2

Clone the repository.

```bash
git clone https://github.com/Anushka25052007/DecodeLab-Python-internship-Anushka
```

---

## Step 3

Navigate to the required project folder.

Example:

```bash
cd Project1
```

or

```bash
cd Project2
```

---

## Step 4

Run the Python file.

Example:

```bash
python project1-todo-list.py
```

or

```bash
python project2-expense-tracker.py
```
